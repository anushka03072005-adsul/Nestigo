# 🏡 Nestigo – Vacation Rental Platform

Nestigo is a full-stack web application that allows users to explore, create, and manage vacation rental listings with secure authentication and scalable deployment.

---

## 🚀 Features

- 🔐 User Authentication & Authorization (Passport.js)
- 🏠 Create, Read, Update, Delete (CRUD) property listings
- 🖼️ Image upload and storage using Cloudinary
- ✅ Input validation using Joi
- 🛡️ Security enhancements (Helmet, MongoDB Sanitization)
- 📦 Dockerized application with MongoDB integration
- 🌐 Fully deployed and accessible online

---

## 🛠️ Tech Stack

**Frontend:**
- EJS, HTML, CSS, Bootstrap

**Backend:**
- Node.js, Express.js

**Database:**
- MongoDB (Mongoose)

**Security & Validation:**
- Passport.js, Joi, Helmet, express-mongo-sanitize

**DevOps & Deployment:**
- Docker, Docker Compose, Render
